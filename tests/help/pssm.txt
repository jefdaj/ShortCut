The pssm extension is for PSI-BLAST position-specific substitution matrix as ASCII.

You can create them with these 3 functions:
  psiblast_train : num faa faa -> pssm
  psiblast_train_all : num faa faa.list -> pssm
  psiblast_train_db : num faa faa.blastdb -> pssm

And use them with these 5 functions:
  psiblast_pssm : num pssm faa -> bht
  psiblast_pssm_all : num pssm faa.list -> bht
  psiblast_pssm_db : num pssm faa.blastdb -> bht
  psiblast_pssm_db_each : num pssm faa.blastdb.list -> bht.list
  psiblast_pssm_each : num pssm faa.list -> bht.list
