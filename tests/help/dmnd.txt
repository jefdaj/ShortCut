The dmnd extension is for DIAMOND database.

You can create them with these 2 functions:
  diamond_makedb : faa -> dmnd
  diamond_makedb_all : faa.list -> dmnd

And use them with these 4 functions:
  diamond_blastp_db : num num faa dmnd -> bht
  diamond_blastp_rev : num num faa dmnd -> bht
  diamond_blastp_rev : num num faa dmnd -> bht
  diamond_blastx_db : num num fna dmnd -> bht
