
[num (number in scientific notation), fna (FASTA (nucleic acid)),
 faa (FASTA (amino acid)) encoded as blastdb] -> bht (tab-separated table of blast hits (outfmt 6))

Like blastx, but BLASTs against an existing database.

Inputs:
  an e-value cutoff
  a query FASTA amino acid file
  a subject BLAST protein database

Algorithm:
  BLAST the query sequences against the database

Output:
  a table of all hits below the cutoff
