The gbk extension is for Genbank files.

You can create them with these 2 functions:
  load_gbk : str -> gbk
  load_gbk_path : str -> gbk

And use them with these 4 functions:
  gbk_to_faa : str gbk -> faa
  gbk_to_faa_rawids : str gbk -> faa
  gbk_to_fna : str gbk -> fna
  gbk_to_fna_rawids : str gbk -> fna
