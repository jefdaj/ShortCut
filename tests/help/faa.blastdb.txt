The faa.blastdb extension is for FASTA amino acid encoded as NCBI BLAST+ sequence database.

You can create them with these 4 functions:
  blastdbget_faa : str -> faa.blastdb
  load_faa_db : str -> faa.blastdb
  makeblastdb_prot : faa -> faa.blastdb
  makeblastdb_prot_all : faa.list -> faa.blastdb

And use them with these 8 functions:
  blastp_db : num faa faa.blastdb -> bht
  blastx_db : num fna faa.blastdb -> bht
  psiblast_db : num faa faa.blastdb -> bht
  psiblast_each_pssm_db : num pssm.list faa.blastdb -> bht.list
  psiblast_pssm_db : num pssm faa.blastdb -> bht
  psiblast_pssms_db : num pssm.list faa.blastdb -> bht
  psiblast_train_db : num faa faa.blastdb -> pssm
  psiblast_train_pssms_db : num faa.list faa.blastdb -> pssm.list
