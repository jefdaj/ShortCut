BLAST one file of nucleic acid (DNA) sequences against another.

Given:
  an e-value cutoff
  a query FASTA nucleic acid file
  a subject FASTA nucleic acid file

It:
  makes a nucleotide database from the subject sequences
  BLASTS the query sequences against the database
  returns a table of all hits below the cutoff
