A scripting language that makes short work of phylogenomic cuts.

Usage:
  shortcut --help | --version
  shortcut [options]

Options:
  -i, --interactive           Start the interpreter, and load SCRIPT if any.
                              This is the default if no arguments are given.
  -s SCRIPT, --script SCRIPT  ShortCut script to run, or load in the REPL
  -w WDIR, --workdir WDIR     Working directory and start point for
                              relative paths in the script [default: .]
  -c CDIR, --cachedir CDIR    Where to cache results, including many
                              intermediate files [default: _shortcut]
  -h, --help                  Print this help text and exit
      --version               Print version number and exit
  -v, --verbose               Print debugging messages to stderr
  -t, --test                  Run self-tests before doing anything else
