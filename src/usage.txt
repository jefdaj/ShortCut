A scripting language that makes short work of phylogenomic cuts.

Usage:
  shortcut --help | --version | --test
  shortcut [options]

Options:
  --help                    Print this help text and exit.
  --version                 Print version number and exit.
  --debug                   Print debugging messages to stderr
  --test                    Run self-tests and exit.
  --interactive             Start the interpreter, and load SCRIPT if any.
                            This is the default if no arguments are given.
  --script SCRIPT           ShortCut script to run, or load with --interactive
  --tmpdir TDIR             Where to cache results, including many large
                            intermediate files.
                            [default: ~/.shortcut]
  --config CONFIG           Load any/all of these options from a config file.
                            [default: ~/.shortcut/config]
  --workdir WDIR            Working directory and start point for relative paths.
                            [default: .]
  --clusterscript CSCRIPT   Script to run arbitrary commands on the cluster.
                            ShortCut makes sure only LIMIT instances are
                            running at the same time, but otherwise assumes
                            this script will handle concurrency.
  --clusterlimit LIMIT      Max number of CSCRIPTs to run at once.
                            If zero, no limit is set. [default: 0]
